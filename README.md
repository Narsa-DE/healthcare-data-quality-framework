# Healthcare Data Quality Framework

Production-grade EHR data ingestion and quality validation pipeline.

## Tech Stack
- Apache Airflow 2.7
- AWS Glue + S3
- Snowflake + dbt
- Python 3.10

## Results
- 99% data accuracy
- 35% reduction in data quality errors
- 5M+ member records processed daily
- Supports HL7 v2, CCD, FHIR R4
